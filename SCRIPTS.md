# 脚本说明

本项目包含以下实用脚本:

## 1. run.sh
主程序运行脚本
- 功能: 编译并运行 Qt RTSP 监控程序
- 用法: `./run.sh`

## 2. monitor_performance.sh  
性能监控脚本
- 功能: 实时监控程序的 CPU、内存使用率
- 用法: `./monitor_performance.sh`
- 说明: 在程序运行时使用,可以查看资源占用情况

## 3. debug_rga.sh
RGA 硬件加速调试脚本
- 功能: 运行程序并显示 RGA 硬件加速的详细调试信息
- 用法: `./debug_rga.sh`
- 说明: 
  - 仅输出前5帧的详细调试信息
  - 用于诊断 RGA 硬件加速是否正常工作
  - 如果 RGA 成功,会显示 "✅ RGA硬件转换连续成功"
  - 如果 RGA 失败,会显示错误码和详情

## 使用建议

1. **首次运行**: 使用 `./run.sh` 启动程序
2. **性能测试**: 运行程序后,另开终端执行 `./monitor_performance.sh`
3. **RGA 问题排查**: 如果视频卡顿,使用 `./debug_rga.sh` 检查硬件加速状态
